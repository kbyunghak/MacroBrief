# Step M5 Store Release Readiness v1

## Goal
Prepare required compliance and operational artifacts for app store publishing.

## Required Artifacts
- Privacy policy draft
- Terms of use draft
- Risk disclaimer copy (information-only)
- Support contact and escalation path
- App metadata (name, subtitle, screenshots, keywords)

## Store Policy Focus
- No financial advice claims
- No misleading performance claims
- Clear user data handling disclosure
- Consent and deletion path for user data

## Operational Readiness
- Crash reporting tool selection
- Basic analytics event plan
- Release checklist (build, QA, metadata, submission)

## Exit Criteria
- Policy docs drafted
- Store listing assets checklist complete
- Release checklist ready
