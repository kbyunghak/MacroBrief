# Step M6 Mobile Beta Rollout Plan v1

## Goal
Run controlled mobile beta before public launch.

## Channels
- iOS: TestFlight
- Android: Closed testing track

## Beta Cohort
- 30-100 users
- Mixed holding profiles
- Mix of heavy/light daily users

## KPI Set
- Crash-free sessions >= 99%
- D7 retention >= 20%
- Notification open rate baseline + trend
- Relevance positive feedback >= 70%

## QA Focus
- Session restore reliability
- Alert feed freshness
- Holdings sync correctness
- Battery/network impact sanity check

## Weekly Cadence
- Week 1: stability baseline
- Week 2: relevance tuning
- Week 3: notification tuning
- Week 4: release candidate validation

## Exit Criteria
- KPI thresholds met or clear remediation plan
- Known critical bugs resolved
